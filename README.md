# Hi, I'm Jose Fuentes

I am a Ph.D. Candidate in Computer Science and Applied Mathematician specializing in Computer Vision and Deep Learning for the Physical World. My work focuses on building perception systems that are mathematically rigorous, physically consistent, and deployment-ready.

## 🔬 Core Research & Engineering Pillars

- **Multimodal Perception & Autonomous Systems**: Developing frameworks that leverage Vision-Language Models (VLMs) and computer vision to enhance autonomous vehicle autonomy, navigation, localization, and behavior forecasting in real-world environments.

- **Geometric & Bio-inspired Vision**: Created a high-performance Hexagonal Image Processing library in C++/Python to improve angular resolution and feature extraction on non-standard sensor grids.

- **Probabilistic Modeling**: Leveraging Gaussian Processes (GP) and Particle Filters for state estimation and spatial-temporal mapping in high-uncertainty environments.

## 🏆 Selected Impact

- **ICRA 2025 Best Paper Finalist**: Recognized for "Learning-based Adaptive Navigation for Scalar Field Mapping."

- **Optimized Inference**: Deployed resource-efficient ML models on NVIDIA Jetson and Edge hardware, achieving stable 1 Hz real-time inference in complex field deployments.

- **Robustness**: Reduced human intervention by 50% in autonomous missions by grounding Reinforcement Learning agents in Partial Differential Equations (PDE).
## 🚀 Featured Projects

### Autonomous Robotics & Field Mapping
- **[Surveyor Gradient Tracking](https://github.com/FIU-MoRA-Lab/surveyor_gradient_tracking)** - ICRA 2025 finalist implementation: Learning-based adaptive navigation algorithm for scalar field mapping on Autonomous Surface Vehicles (ASVs).
- **[SeaRobotics Surveyor Control](https://github.com/FIU-MoRA-Lab/searobotics_surveyor)** - Control library for ASV SeaRobotics platform with sensor integration, IMU, and high-level autonomous control capabilities.
- **[BlueROV Control Library](https://github.com/FIU-MoRA-Lab/blue_rov)** - Python-based control interface for BlueRobotics underwater vehicle with comprehensive sensor and actuation support.

### Physics-Informed AI & PDE Solvers
- **[TidalFlow-SWE](https://github.com/Xioeng/TidalFlow-SWE)** - PDE solver for shallow water equations with applications to tidal simulation and environmental modeling.
- **[PhysRAG-SWE](https://github.com/Xioeng/PhysRAG-SWE)** - Physics-informed framework for shallow water equation modeling and analysis.
- **[Advection-Diffusion Simulator](https://github.com/Xioeng/advection_diffusion_simulator)** - FEniCS-based numerical solver for advection-diffusion processes, demonstrating high-performance PDE computation.

### Geometric & Computational Methods
- **[Hexagonal Image Processing (HexMAV)](https://gitlab.com/ricardogutic/inv-ing-3497-hexmav)** - High-performance C++/Python library for processing hexagonally sampled images, improving angular resolution and feature extraction on non-standard sensor grids.
- **[C++ Graph & Monte Carlo Library](https://github.com/Xioeng/C-code)** - High-performance implementations of graph algorithms and Monte Carlo simulations with applications to hexagonal grid processing.

### Applications
- **[PawPal](https://github.com/Xioeng/pawpal)** - Streamlit-based pet owner activity scheduler demonstrating full-stack web application development.
