# 💫 About Me:
I am a Ph.D. Candidate in Computer Science and Applied Mathematician specializing in Computer Vision and Deep Learning for the Physical World. My work focuses on building perception systems that are mathematically rigorous, physically consistent, and deployment-ready.<br>🔬 Core Research & Engineering Pillars<br><br>    Multimodal Perception: Participating in the development of Frameworks to improve autonomous vehicle navigation, localization and mapping.<br><br>    Geometric & Bio-inspired Vision: Created a high-performance Hexagonal Image Processing library in C++/Python to improve angular resolution and feature extraction on non-standard sensor grids.<br><br>    Probabilistic Modeling: Leveraging Gaussian Processes (GP) and Particle Filters for state estimation and spatial-temporal mapping in high-uncertainty environments.<br><br>🏆 Selected Impact<br><br>    ICRA 2025 Best Paper Finalist: Recognized for "Learning-based Adaptive Navigation for Scalar Field Mapping."<br>    Optimized Inference: Deployed resource-efficient ML models on NVIDIA Jetson and Edge hardware, achieving stable 1 Hz real-time inference in complex field deployments.<br><br>    Robustness: Reduced human intervention by 50% in autonomous missions by grounding Reinforcement Learning agents in Partial Differential Equations (PDE).
